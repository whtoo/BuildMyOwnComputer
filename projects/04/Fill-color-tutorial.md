问题：实现一个程序，（定期）检查键盘是否有按键按下。如果有，则在屏幕当前位置画一个黑色像素；否则，请擦除当前位置的像素，并将绘制位置回退一格直到屏幕左上角原点。



分析：

首先，我们得搞清楚3件事情。

1. 如何获取键盘输入；
2. 绘制平面是什么；
3. 如何在当前位置绘制一个像素；


解决：

1. ```asm
   (LOOP)
   @KBD
   D=M    // 将键盘输入从寄存器转存到数据寄存器

   @BLACK
   D;JNE  // 如果，数据寄存器中非零值，则执行黑色像素绘制

   @SCREEN
   D=M

   @WHITE
   D;JNE

   @LOOP
   0;JMP
   ```
